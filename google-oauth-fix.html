🔧 Sửa lỗi Google OAuth: redirect_uri_mismatch

Hướng dẫn chi tiết để khắc phục lỗi đăng nhập Google

❌ Lỗi hiện tại:

Error 400: redirect_uri_mismatch

Chi tiết: redirect_uri=http://localhost:3997/auth/google/callback

Ứng dụng không tuân thủ chính sách OAuth 2.0 của Google

✅ Giải pháp:

Cần đăng ký URI chuyển hướng trong Google Cloud Console

URI cần thêm: http://localhost:3999/auth/google/callback

Bước 1: Truy cập Google Cloud Console

Mở trình duyệt và truy cập:

https://console.cloud.google.com/
Mở Google Cloud Console

Bước 2: Chọn Project

Chọn project của bạn hoặc project chứa OAuth 2.0 credentials

💡 Gợi ý: Tìm project có tên liên quan đến ứng dụng học tiếng Hàn của bạn

Bước 3: Vào APIs & Services → Credentials

Trong menu bên trái:

  • Nhấp vào "APIs & Services"
  • Chọn "Credentials"
📍 Menu: APIs & Services → Credentials

Bước 4: Tìm OAuth 2.0 Client

Tìm OAuth 2.0 Client ID có thông tin:

Client ID: 875597054714-e1v4icg0gisrliunj1tt499f7n66lmke.apps.googleusercontent.com

Nhấp vào tên Client ID để chỉnh sửa

Bước 5: Thêm Authorized redirect URIs

Trong phần "Authorized redirect URIs", thêm URI sau:

http://localhost:3999/auth/google/callback
⚠️ Quan trọng:
  • Đảm bảo port là 3999
  • Không có dấu "/" ở cuối
  • Sử dụng "http" chứ không phải "https" cho localhost

Bước 6: Lưu thay đổi

Nhấp nút "SAVE" để lưu cấu hình

⏱️ Lưu ý: Có thể mất 5-10 phút để thay đổi có hiệu lực

Bước 7: Kiểm tra cấu hình server

Đảm bảo server đang chạy trên port 3999:

Port hiện tại: 3999 Callback URL: http://localhost:3999/auth/google/callback
⏳ Đang kiểm tra trạng thái server...

Bước 8: Test đăng nhập

Sau khi hoàn thành các bước trên:

Test đăng nhập Google Test Google OAuth trực tiếp

🔄 Tài khoản test backup (nếu Google OAuth vẫn lỗi):

  • Admin: admin@test.com / admin123
  • Teacher: teacher@test.com / teacher123
  • Student: student@test.com / student123

🔍 Troubleshooting

Nếu vẫn gặp lỗi:

  • Xóa cache trình duyệt
  • Thử ở chế độ incognito
  • Kiểm tra lại URI trong Google Console
  • Đợi 10-15 phút sau khi lưu trong Google Console
